# Executando Zuul a partir dos fontes
1. Acesse a pasta exemplo-zuul
2. Configure as regras de rota no arquivo application.yml
3. Execute o seguinte comandos
   * No Linux
    ```bash
    $ ./gradlew bootRun
    ```
   * No Windows
